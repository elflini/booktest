2.2 Pipe : %>%

  • Magrittr 방식을 의미하여, pipe라고 부른다.
  • R-studio에서 pipe를 간단하게 부르는 방법은 Ctrl + Shift + M 단축키를 이용한다.
  • Pipe는 순서대로 명령어를 전달하는 역할을 하며, 직관적으로 명어를 구성하기에 용이하다. 아래 예제는 mpg 데이터를 filter() 함수로 보내어 명령을 시행한다.
mpg %>% 
  filter(year > 2000 & class %in% c("subcompact"))

2.4 group_by()와 summarise()

  • R에서 기본적인 연산은 행 단위로 이루어지게 된다. 예를 들어, 변수 a와 변수 b를 더하는 경우를 생각해보자. 이 경우 첫 번째 행의 a값과 b값을 더하고 계속해서 두 번째 행의 a값과 b값을 더하는 식으로 결과를 보여주게된다.
  • group_by() 함수는 행단위의 연산이 아닌 열 단위의 연산이 가능하게 해준다. 예를 들어 id가 1인 행이 5개가 있다고 했을때 a 변수의 값을 group_by()를 통해 더해주게 되면 id 별로 연산해서 결과를 산출한다. 즉, 기본적으로 하나의 열 내에서 여러개의 행에 해당하는 값을 연산하고자 할 때 사용한다.
  • group_by() 함수는 단독으로 사용하면 별다른 행동을 취하지 않기 때문에 summarise() 함수와 같이 사용한다.
