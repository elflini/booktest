7.1 하나의 변수에 의한 관찰값의 정렬

데이터를 정렬하기 위해서는 정렬을 위한 기준 변수가 있어야 하며, 먼저 기준 변수가 하나인 경우부터 살펴보자. 정렬을 위해 order() 함수 또는 arrange() 함수를 사용가능하며, 사용 방법은 다음과 같다.

정렬된 데이터 <- 데이터[order(기준 변수),] 또는
정렬된 데이터 <- 데이터 %>% arrange(기준 변수)

정렬은 기본적으로 오름차순으로 진행되며, 내림차순으로 데이터를 정렬하려면 다음과 같이 사용한다.

정렬된 데이터 <- 데이터[order(기준 변수, decreasing=T),] 또는
정렬된 데이터 <- 데이터 %>% arrange(desc(기준))

t20 데이터셋에서 심결요양급여비용총액(ed_rc_tot_amt) 변수를 기준으로 오름차순으로 정렬시켜 t20_sort 데이터셋에 저장해보자. 이를 위한 프로그램은 다음과 같다.

7.2 여러 개의 변수에 의한 관찰값의 정렬

데이터 정렬 시 기준 변수가 하나가 아닌 여러 개의 변수를 사용할 때, 정렬 방법은 다음과 같다.

정렬된 데이터 <- 데이터[order(기준 변수1, 기준 변수2),] 또는
정렬된 데이터 <- 데이터 %>% arrange(기준 변1수, 기준 변수2)

오름차순과 내림차순은 decreasing 옵션에서 TRUE, FALSE값을 벡터로 주거나, desc() 함수를 변수별로 취사선택해서 사용한다. 예를 들어, 첫번째 변수로는 오름차순으로 정렬하고, 두 번째 변수로는 내림차순으로 정렬하는 경우에 다음과 같이 사용한다.

정렬된 데이터 <- 데이터[order(기준 변수1, 기준 변수2, decreasing=c(F,T),] 또는
정렬된 데이터 <- 데이터 %>% arrange(기준 변수1, desc(기준 변수2))

여러 변수를 기준으로 데이터 정렬시 정렬 순서는 기준 변수1을 기준으로 먼저 데이터를 정렬한다. 그 후 같은 값을 가지는 개체가 여러 개 있다면 그 안에서 기준 변수2를 기준으로 정렬하며, 같은 값을 가지는 개체가 있다면 기준 변수 3을 기준으로 다시 세부적으로 정렬한다.

t20 데이터셋에서 개인고유번호(rn_indi)로 자료를 오름차순으로 정렬하고, 같은 값이 존재하는 경우 심결요양급여비용총액(ed_rc_tot_amt) 변수를 기준으로 오름차순으로 정렬시켜 t20_sort2 데이터셋에 저장해보자. 이를 위한 프로그램은 다음과 같다.

7.3 처음과 마지막 개체 추출하기

자료를 ID 변수를 기준으로 자료를 정렬한 후 ID 별로 첫번째 개체와 마지막 개체를 추출하고 싶을때 row_number() 활용하며, 그 방법은 다음과 같다.

# ID별 첫번째 자료
first_data <- data %>% arrange(rn_indi) %>% group_by(ID) %>% filter(row_number()==1)

# ID별 마지막 자료
last_data <- data %>% arrange(rn_indi) %>% group_by(ID) %>% filter(row_number()==n())

다음과 같은 예제를 살펴보자.

t20_sort_first <- t20 %>% 
  arrange(rn_indi) %>% 
  group_by(rn_indi) %>% 
  mutate(first=ifelse(row_number()==1,1,0),
         last=ifelse(row_number()==n(),1,0))

7.4 동일한 관찰값의 제거

동일한 개체가 중복되어 나타나는 경우 distinct() 함수를 이용하여 중복 개체를 제거할 수 있다.

새로운 데이터 <- 데이터 %>% distinct()

예제를 위해, t40 데이터셋으로부터 요양개시일자(mdcare_strt_dt)를 이용해 날짜변수(date1)를 생성하고, 요양급여상병기호(mcex_sick_sym)를 첫 두 자리만 추출하여 sick 변수를 생성한 다음 t40_nodup 데이터셋을 생성해보자. 이때 rn_key, date1, sick 변수만 남기자.

t40_nodup <- t40 %>% 
  mutate(date1=ymd(mdcare_strt_dt), sick=substr(mcex_sick_sym,1,2)) %>% 
  select(rn_key, date1, sick) %>% 
  arrange(rn_key, date1)

만약 각 개체가 가지는 모든 변수의 값이 같은 경우가 아니라, 정렬에 사용된 기준 변수의 값이 같은 경우 중복된 개체를 제거하고 싶다면 distinct() 함수의 옵션으로 기준 변수를 넣어주기만 하면 되며, 사용방법은 다음과 같다.

새로운 데이터 <- 데이터 %>% 
    distinct(기준 변수1, 기준 변수2, ... , .keep_all=T)
