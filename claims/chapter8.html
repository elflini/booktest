Chapter 8 반복문

공단자료와 필요 패키지 불러오기

load("claim.RData")
library(tidyverse)
library(lubridate) # 날짜 함수

일정한 조건하에서 정의된 명령문의 반복 수행을 지시하는 반복문의 종류는 다음과 같다.

* for (반복) 명령문
* while (조건) 명령문
* repeat 명령문

for() 문은 정의된 반복 형태에 따라 명령문을 수행하며, while() 문은 조건이 참인 경우에만 명령문을 수행한다. repeat() 문은 조건 없이 명령문을 반복 수행하는 것으로 무한 반복 수행을 막기 위해서 무조건 분기문을 반드시 포함해야 한다. for() 문과 while() 문은 조건을 먼저 확인한 후 명령문 수행 여부를 결정하지만 repeat() 문은 주어진 명령문을 먼저 수행한다. 따라서 명령문의 순서에 따라 동일한 내용의 프로그램이라도 결과가 다를 수 있다.

데이터 전처리 과정에서 반복문은 일반적으로 반복 수행에 대한 조건을 알고 시행하는 것이기 때문에 for() 문을 이용한 반복이 적합하며, 따라서 여기서는 for 문을 이용하여 데이터 전처리 과정의 예를 살펴본다. for 문의 기본적인 사용법은 다음과 같다.

for (variable in vector) {반복 실행}

여기서 item은 지수(index)에 사용할 임의의 변수이며, vector는 반복횟수를 결정지을 백터 객체이다.

데이터 전처리에서 반복문은 주로 조건문과 결합하여 사용되며, 특정 조건을 만족하는 경우 그 조건에서 동일한 형태의 명령어를 시행해야 하는 경우 사용된다.

8.1 조건에 의한 반복

t20 데이터셋에서 기준연도(std_yyyy)가 ‘2002’년일 때 심결요양급여비용총액(ed_rc_tot_amt)에 1%를 더해 totalcost 변수를 생성하고, ’2003’년에는 2% 추가와 같은 식으로 매해 1%씩 더 더해져서 totalcost가 산출된다고 가정하자. 이 경우 ’기준년도’가 바뀜에 따라 비슷한 패턴의 계산이 계속 반복되기 때문에 반복문을 이용할 수 있으며, 방법은 다음과 같다.

t20_for <- t20

8.2 개체에 의한 반복

t20 데이터셋에서 기준연도(std_yyyy)가 ‘2002’년일 때 심결요양급여비용총액(ed_rc_tot_amt)에 1000을 더해 totalcost 변수를 생성하고, 동일 조건에 만약 심결본인부담금(edc_sba)이 3000보다 크다면 0원으로 대체하여 edc_sba2 변수를 생성해보자. 또한 기준연도(std_yyyy)가 ‘2003’년일 때 심결요양급여비용총액(ed_rc_tot_amt)에 2000을 더해 totalcost 변수를 생성하고, 동일 조건에 만약 심결본인부담금(edc_sba)이 3000보다 크다면 2000원으로 대체하여 edc_sba2 변수를 생성해보자. 즉, std_yyyy가 ‘2002’ 일 때 두 가지 작업을 시행하고 마찬가지로 std_yyyy가 ‘2003’ 일 때도 두 가지 작업을 시행한다. 이를 위한 프로그램은 다음과 같다.

t20_temp <- t20 %>% 
  filter(std_yyyy %in% c('2002', '2003', '2004')) %>% 
  select(rn_indi, std_yyyy, rn_key, ed_rc_tot_amt, edc_sba)

t20_temp$totalcost <- t20_temp$edc_sba2 <- NA

for (i in 1:nrow(t20_temp) ) {
  
if (t20_temp[i,"std_yyyy"]=='2002') {
  t20_temp$totalcost[i] <- t20_temp$ed_rc_tot_amt[i] + 1000
if (t20_temp[i,"edc_sba"]>3000) {
  t20_temp$edc_sba2[i] <- 0
}
  }

if (t20_temp[i,"std_yyyy"]=='2003') {
  t20_temp$totalcost[i] <- t20_temp$ed_rc_tot_amt[i] + 2000
if (t20_temp[i,"edc_sba"]>3000) {
  t20_temp$edc_sba2[i] <- 2000
}
  }  
  
  
}

8.3 다중 반복

조건문에서 if 문을 중첩해서 사용하듯 반복문도 중첩해서 사용 가능하며 사용 방법은 다음과 같다.

for (i in 1:N) {
  for (j in 1:K) {
실행문
  }
}

여기서 반복 순서는 첫 번째 반복인 i가 1일때 j가 1부터 K까지 반복이 일어나며, 이러한 반복은 i의 값이 N에 이를 때까지 계속된다.

t20_double <- t20 %>% mutate(year=as.numeric(substr(mdcare_strt_dt,1,4)),
                             month=as.numeric(substr(mdcare_strt_dt,5,6))) %>% select(rn_indi, year, month)

t20_double$yymm <- NA
for (i in 2002:2015) {
  for (j in 1:12) {
   t20_double$yymm[t20_double$year==i & t20_double$month==j] <- paste(i,j,sep="-")
  }
}

i는 2002부터 2015까지 1단위로 증가하면서 값이 바뀌며, 2002부터 2015까지의 크기인 14만큼 반복 수행하게 된다. j는 1부터 12까지 1단위로 증가하면서 값이 바뀌며, 12만큼 반복수행 하게 된다. 이때 반복수행 순서는 i의 값이 바뀔 때마다 j는 전체 반복을 한다. 즉, i=2002일 때 j는 1부터 12까지 반복하며 다시 i=2003일 때 j는 1부터 12까지 반복한다. 다시 말해서 반복 순서는 바깥쪽에서부터 색인의 값이 하나씩 결정되며 제일 안쪽에 있는 for 문은 전체를 반복한다. 안쪽의 for 문이 끝나면 바깥쪽으로 순서대로 for 문을 끝낸다. 프로그램에서 paste 함수는 문자열을 결합할 때 사용하는 함수이다. 따라서 [paste(i,j,sep=“-”)]는 year=i이고 month=j일 때 yymm 변수는 ’i – j’ 값을 가지게 된다. 예를 들어, i=2002, month=12이면, yymm 변수는 ‘2002 – 12’ 라는 값을 가지게 되며 이러한 과정을 2002부터 2015까지, 1월부터 12월까지 모든 조합으로 조건을 검색하여 yymm 변수를 생성한다.
