Chapter 10 Significance Testing

10.1 Hypotheses

  • \(H_0\) and \(H_a\) are statements about probability models or, equivalently, about population characteristics.

  • The null hypothesis, \(H_0\), usually says no effect, no difference, etc.

    • In terms of parameters, it usually is written as \(H_0\) : \(\theta = \theta_0\); \(H_0\) : \(\theta \le \theta_0\); or \(H_0\) : \(\theta ≥ \theta_0\), where \(\theta_0\) is a value specified by the investigator.
    • It is important that the null contains a point of equality.

  • The alternative states that the null is false and usually is written as \(H_a\) : \(\theta \neq \theta_0\); \(H_a\) : \(\theta > \theta_0\); or \(H_a\) : \(\theta < \theta_0\).


10.2 Assessing the Evidence

  • A significance test is a test of \(H_0\). The strategy is as follows:
    • Translate the scientific hypothesis into \(H_0\) and \(H_a\).
    • Begin with the assumption that \(H_0\) is true.
    • Collect data.
    • Determine whether or not the data contradict \(H_0\).
      • The p-value is a measure of how strongly the data contradict the null.
      • A small p-value is strong evidence against \(H_0\).

Test statistic

  • Definition: A test statistic is a function of the data and \(\theta_0\).
    • The test statistic is chosen to discriminate between \(H_0\) and \(H_a\).
    • Usually, it incorporates an estimator of \(\theta\).
    • Familiar test statistics are
      • \((a) \ \ Z = \frac{θ-θ_0}{SE(\hat θ|H_0)}\)
      • \((b) \ \ t = \frac{\bar X-\mu_0}{S_X/ \sqrt n}\)
      • \((c) \ \ X^2 = \frac{(n-1)S^2_X}{σ^2_0}\)

\(p\)-value

  • Denote the test statistic as \(T\) and denote the realized value of \(T\) as \(t_{obs}\).
    • The p-value is a measure of consistency between the data and \(H_0\).
    • It is defined as

\[ p-\text{value} = P(T \text{ is as or more extreme than } t_{obs} \text{ in the direction of } H_a|H_0). \]

  • A small \(p\)-value means that the data are not consistent with \(H_0\).
    • That is, small \(p\)-values are taken to be evidence against \(H_0\).
  • Common Error I: Many investigators interpret a large \(p\)-value to be evidence for \(H_0\).
    • This is not correct. A large p-value means that there is little or no evidence against \(H_0\).
    • For example, consider a test of \(H_0: \mu = 100\) versus Ha : \(\mu \neq 100\) based on a sample of size \(n = 1\) from \(N(0, 20^2)\).
    • Suppose that the true mean is \(\mu = 105\) and that \(X = 101\) is observed.
    • The \(Z\) statistic is

\[ z_{obs} = \frac{101-100}{20} = 0.05 \]

  • And the \(p\)-value is

\[ p-value = 2[1− Φ(0.05)] = 2(1 − 0.5199) = 0.9601 \]

  • The \(p\)-value is large, but the data do not provide strong evidence that \(H_0\) is true.

  • Common Error II: Many investigators interpret a very small \(p\)-value to mean that a large (important) effect has been found.

    • This is not correct.
    • A very small \(p\)-value is strong evidence against \(H_0\).
    • It is not evidence that a large effect has been found.
    • Example: Suppose that the standard treatment for the common cold reduces symptoms in \(62\%\) of the population.
    • An investigator develops a new treatment and wishes to test \(H_0: p = 0.62\) against \(H_a: p > 0.62\). The usual test statistic is

\[ Z=\frac{\hat p− p_0}{\sqrt{\frac{p_0(1-p_0)}{n}}} \]

where \(p_0 = 0.62\).

  • If the new treatment reduces symptoms in \(62.1\%\) of a sample of size \(n = 3,000,000\), then the observed value of the test statistic is

\[ z_{obs}=\frac{.621-.62}{\sqrt{\frac{.62(1-.62)}{3,000,000}}}= 3.5684 \]

  • The \(p\)-value is \(P(Z > 3.5684|H_0) = 0.00018\).
    • This is strong evidence against \(H_0\), but the effect size is very small.
  • Using likelihood to find a test statistic.
    • One way to find a test statistic for testing \(H_0 : \theta = \theta_0\) against a one or two-sided alternative is to examine the likelihood ratio

\[ LR=\frac{L(θ_0|X)}{\underset{θ}{max}L(θ|X)} \]

where the maximization in the denominator is over all \(\theta\) that satisfy \(H_a\).

  • The LR is ratio of the probability of the data under \(H_0\) to the largest possible probability of the data under \(H_a\).
    • The LR satisfies \(LR \in (0, 1)\).
    • Small values of LR are interpreted to be evidence against \(H_0\).
    • Example: suppose that \(H_0: p = p_0\) is to be tested against \(H_a: p \ne p_0\) using a random sample from \(Geom(p)\).
    • Recall that the MLE of \(p\) is \(\hat p=1/ \sqrt X\).
    • The likelihood ratio is

\[ LR = \frac{(1-p_0)^{n(\bar X-1)}p^n_0}{(1-1/\sqrt X)n \bar X^{-n}}=(\frac{1-p_0}{\bar X-1})^{n(\bar X-1)} \bar X^{n\bar X}p^n_0 \]

  • In the following display, the log of the LR statistic is plotted against \(\bar X\) for the special case \(n = 10\) and \(p_0 = 0.25\)


  • The above plot reveals that the log likelihood ratio statistic is small if \(\bar X\) is substantially larger or substantially smaller that 4.
    • Note that \(\bar X = 4 \Longrightarrow 1/\bar X = 0.25 = p_0\) and that the LR statistic is 1 if \(\bar X = 4\); i.e., the log of the LR statistic is zero.
  • To use the likelihood ratio as a test statistic, it is necessary to determining its sampling distribution under a true null.
    • This can be quite difficult, but fortunately there is an easy to use large sample result.
    • If \(\theta\) is a scalar, then under certain regularity conditions, the asymptotic null distribution of \(−2 ln(LR)\) is \(\chi^2_1\).


10.3 One Sample \(Z\) Tests

  • Form of the test Statistic.
    • Suppose that it is of interest to test \(H_0: \theta = \theta_0\) against either \(H_a: \theta \ne \theta_0\), \(H_a: \theta > \theta_0\), or \(H_a: \theta < \theta_0\).
    • Further, suppose that we have an estimator of \(\theta\) that satisfies

\[ T|H_0 \sim N(θ,\frac{ω^2_0}{n}) \]

and an estimator of \(ω^2_0\), say \(W^2_0\) that satisfies \(W^2_0 \xrightarrow[]{prob} ω^2_0\) whenever \(H_0\) 0 is true.

  • The subscript on \(ω\) is a reminder that the variance of \(T\) under \(H_0\) could be a function of \(\theta_0\).

  • A reasonable test statistic is

\[ Z = \frac{T − θ_0}{W_0/\sqrt{n}} \]

  • If sample size is large, then the distribution of \(Z\) under \(H_0\) is approximately \(N(0, 1)\).

  • \(p\)-values: Let \(z_{obs}\) be the observed test statistic. Then the \(p\)-value for testing \(H_0\) against \(H_a\) is

    • \(1 − P(−|z_{obs}| ≤ Z ≤ |z_{obs}|) =2 [1 − \Phi(|z_{obs}|)]\) if the alternative hypothesis is \(H_a : \theta \neq \theta_0\);
    • \(P(z_{obs} ≤ Z) = 1 − \Phi(z_{obs})\) if the alternative hypothesis is \(H_a : θ > θ_0\); and
    • \(P(Z ≤ z_{obs}) = \Phi(z_{obs})\) if the alternative hypothesis is \(H_a : \theta < \theta_0\).

  • Example: \(X_1, X_2, \ldots , X_n\) is a random sample from a population with mean \(\mu\) and variance \(\sigma^2\).

    • To test \(H_0: \mu = \mu_0\) against either \(H_a: \mu \neq \mu_0\), \(H_a: \mu > \mu_0\), or \(H_a: \mu < \mu_0\), use the test statistic

\[ Z = \frac{\bar X − µ_0}{S_X/\sqrt{n}} \]

  • Example: \(X_1, X_2,\ldots , X_n\) is a random sample from \(Bern(p)\).
    • To test \(H_0: p = p_0\) against either \(H_a: p \neq p_0\), \(Ha: p > p_0\), or \(Ha: p < p_0\), use the test statistic

\[ Z = \frac{\bar p− p_0}{\sqrt{p_0(1-p_0)/n}} \]

  • Note that \(ω^2_0 = p_0(1 − p_0)\).


10.4 One Sample \(t\) Tests

  • Suppose that \(X_1, X_2, \ldots , X_n\) is a random sample from \(N(\mu, \sigma^2)\).
    • To test \(H_0: \mu = \mu_0\) against either \(H_a: \mu \ne \mu_0\), \(H_a: \mu > \mu_0\), or \(H_a: \mu < \mu_0\), use the test statistic

\[ T = \frac{\bar X − \mu_0}{S_X/\sqrt{n}} \]

  • Under \(H_0\), the test statistic has a \(t\) distribution with \(n − 1\) degrees of freedom.


10.5 Some Nonparametric Tests

  • We will examine only the sign test.
    • Suppose that \(X_1, X_2, \ldots , X_n\) is a random sample from a continuous distribution having median \(\tilde \mu\).
    • A test of \(H_0: \tilde \mu = \tilde \mu_0\) against either \(H_a: \tilde \mu \ne \tilde \mu_0\), \(H_a: \tilde \mu > \tilde \mu_0\), or \(H_a: \tilde \mu < \tilde \mu_0\) is desired.
    • Let

\[ U_i = = I_{(−\infty,\tilde \mu_0]}(X_i) = \begin{cases} 1 & if \ X_i ≤ \tilde \mu_0; \ and \\ 0 & otherwise. \end{cases} \]

  • Under \(H_0\), \(U_i\sim iid Bern(0.5)\) and \(Y = \sum_{i=1}^{n} 1 \ Ui_ \sim Bin(n, 0.5)\).

  • Accordingly, the test statistic is distributed approximately \(N(0, 1)\) under \(H_0\), where \(\hat p = Y/n\).

\[ Z = \frac{\hat p− 0.5}{\sqrt{.25/n}} \]


10.6 Probability of the Null Hypothesis

  • When using the frequentist approach, it is not correct to interpret the \(p\)-value as the probability that \(H_0\) is true.
    • When using the Bayesian approach, then the posterior probability that \(H_0\) is true can be computed.
  • Suppose, for example, that \(X_i\sim iid \ Bern(\theta)\) for \(i = 1,\ldots , n\).
    • Furthermore, suppose that the prior on \(\theta\) is \(\theta\sim Beta(\alpha_1, \alpha_2)\).
    • The sufficient statistic is \(Y =\sum_{i=1}^{n} X_i\) and the posterior is \(\theta|(Y = y) \sim Beta(\alpha_1 + y, \alpha_2 + n − y)\).
    • Suppose that one wants to test \(H_0: \theta ≤ \theta_0\) against \(H_a: \theta > \theta_0\). Then

\[ P(H_0|Y = y) = P(\theta ≤ \theta_0|Y = y) =\int_{0}^{\theta_0} g_{\theta|Y} (\theta|y) d\theta \]

  • For example, if \(\alpha_1 = \alpha_2 = 1\), \(n=40\), \(y=30\), and we want to test \(H_0: \theta ≤ 0.6\) against \(H_a: \theta > 0.6\), then \(\theta|(Y = 30)\sim Beta(31, 11)\) and

\[ P(H_0|Y = 30) =\int_{0}^{0.6} \frac {\theta^{30}(1 − \theta)^{10}}{B(31, 11)}d\theta = 0.0274 \]

  • The test statistic for computing the frequentist \(p\)-value is

\[ z_{obs} = \frac{0.75-0.6}{\sqrt{\frac{0.6(1-0.6)}{40}}}=1.9365 \]

  • The p-value is

\[ p-value = P(Z > 1.9365) = 0.0264 \]

  • If the correction for continuity is employed, then

\[ z_{obs} = \frac{0.75-\frac{1}{80}-0.6}{\sqrt{\frac{0.6(1-0.6)}{40}}}=1.7751 \ and \ p-value= 0.0379 \]

  • There are some complications if one wants a Bayesian test of \(H_0: \theta = \theta_0\) against \(H_a: \theta \ne \theta_0\).
