Chapter 6 R 제어문 및 사용자 정의 함수

R에서 제공되는 연산이나 함수 이외에 조건문, 반복문 등을 이용한 프로그래밍을 통해 다양한 형태의 출력결과를 얻을 수 있는 프로그램을 작성할 수 있다. 사용자가 자주 사용하는 프로그램의 경우는 함수와 관련된 인수를 정의하여 선언함으로써 간단히 적용할 수 있으며, 이러한 함수를 사용자 정의 함수라고 한다.

6.1 조건문

R의 조건문과 조건문 기능을 하는 함수로는 다음과 같은 종류가 있다.

  • if (조건) 명령문
  • if (조건) 명령문1 else 명령문2
  • ifelse(조건, 명령문1, 명령문2)
  • switch(기준, 조건1=명령문1, 조건2=명령문2, …)

6.1.1 if()

if() 함수가 사용되는 방법은 3가지가 있다.

  1. if (조건)이 참(TRUE)이면 해당 명령어 실행

  2. else 문을 써서 조건이 참(TRUE)이 아니면 else 다음에 있는 명령어를 실행

  3. else if를 써서 조건을 반복

if (조건) {조건이 만족되면 실행}
if (조건) {조건이 만족되면 실행} else {조건이 만족되지 못하면 실행}
if (조건1) {조건1 만족 실행} else if (조건2) {조건2 만족 실행} else {불만족 실행}
x <- 1:10
if (x[4]==4) {
  print(exp(x[4]))
}
## [1] 54.59815

if (x[4]==4)는 만약 x 벡터의 4번째 원소가 4라는 값을 가진다면, print(exp(x[4])) x 벡터의 4번째 값을 지수변환(exp())하여 출력하라.

if (x[4]==3) {
  print(exp(x[4]))
} else {
  x[4] <- 3
}
x
##  [1]  1  2  3  3  5  6  7  8  9 10

if (x[4]==3) 만약 x 벡터의 4번째 원소가 3이라면 print(exp(x[4])) x 벡터의 4번째 값을 지수변환(exp())하여 출력하고 else x[4]<-3 만약 x 벡터의 4번째 원소가 3이 아니라면 x의 4번째 자리에 3을 집어 넣어라.

# score가 90 이상이면 A, ..., 60 이상이면 D, 나머지(60미만)는 F
score <- 88
if (score >= 90) {
  print("A")
} else if (score >= 80) {
  print("B")
} else if (score >= 70) {
  print("C")
} else if (score >= 60) {
  print("D")
} else {
  print("F")
}
## [1] "B"

조건이 2개 이상이 있는 경우나 조건결과를 반대로 할 경우에는 논리연산자나 부정연산자를 사용한다. &는 그리고의 의미가 있고, |는 또는의 의미가 있으며, !는 조건 결과를 반대로 나타낸다.

Table 6.1: 조건에 사용하는 논리 연산자
연산자 의미 수식예
& AND x > 3 & x < 7
| OR x < 3 | x > 7
! NOT !(x > 3) 
x <- 25
y <- 75
if (x > 0 & y < 100) {
  print("TRUE 입니다")
} else {
  print("FALSE 입니다")
}
## [1] "TRUE 입니다"

6.1.2 ifelse()

ifelse() 함수는 엑셀의 if() 함수와 사용법이 같다. 즉, 함수의 첫번째 인수인 조건이 참이면 두번째 인수를 실행하고 그렇지 않으면 3번째 인수를 실행한다.

ifelse(조건, 조건이 만족되면 실행, 조건이 만족되지 못하면 실행)
x <- 3
ifelse(x %% 2 == 0, "짝수", "홀수")
## [1] "홀수"

ifelse()는 벡터 데이터도 검사해서 그 결과를 벡터로 출력한다. if() 함수는 조건식의 값이 TRUE 또는 FALSE 값 하나에 대해서만 작동을 하지만 ifelse()는 TRUE, FALSE의 값이 벡터로 나타나는 경우에도 원소별로 조건을 적용하여 명령어를 실행한다.

x <- c(1, 2, NA, 4, 5, NA, 7)
ifelse(x %% 2 == 0, "짝수", "홀수")
## [1] "홀수" "짝수" NA     "짝수" "홀수" NA     "홀수"

6.1.3 switch()

switch()는 키워드를 조건으로 하여 키워드가 맞는 위치의 명령어를 실행하는 함수이다. 적용해야 하는 키워드와 실행 함수가 많은 경우에 유용하다.

switch(키워드, 키워드1 = 실행1, 키워드2 = 실행2, 키워드3=...)
x <- 1:10
type <- "mean"
switch(type, mean=mean(x), sd=sd(x), median=median(x), range(x))
## [1] 5.5

키워드는 직접 텍스트로 입력하지 않고 해당 순서에 맞는 숫자를 입력해도 실행된다.ㅊ

x <- 1:10
type <- 2
switch(type, mean=mean(x), sd=sd(x), median=median(x), range(x))
## [1] 3.02765

6.2 반복문

일정한 조건하에서 정의된 명령문의 반복 수행을 지시하는 R의 반복문의 종류는 다음과 같다.

  • for (반복) 명령문
  • while (조건) 명령문
  • repeat 명령문

for() 문은 정의된 반복 형태에 따라 명령문을 수행하는 것이고 while() 문은 조건이 참인 경우에만 명령문을 수행한다. repeat() 문은 조건 없이 명령문을 반복 수행하는 것으로 무한 반복 수행을 막기 위해서 무조건 분기문을 반드시 포함해야 한다. for() 문과 while() 문은 조건을 먼저 확인한 후 명령문 수행 여부를 결정하지만 repeat() 문은 주어진 명령문을 먼저 수행한다. 따라서 명령문의 순서에 따라 동일한 내용의 프로그램이라도 결과가 다를 수 있다.

6.2.1 for()

for() 함수는 반복횟수를 지정해 줄 수 있는 경우 사용이 용이하다.

for (item in vector) {반복 실행}

반복횟수는 vector의 원소 갯수만큼 적용된다. 예를 들면 1에서 10까지 있는 벡터가 있다면, 반복횟수는 원소 개수인 10번으로 결정되고 이때 index를 담당하는 item은 반복이 진행됨에 따라 vector의 값인 1, 2, … 순으로 값을 저장하여 사용가능하게 된다.

# 1부터 5까지 출력
for (i in 1:5) {
  print(i)
}
## [1] 1
## [1] 2
## [1] 3
## [1] 4
## [1] 5
# 알파벳 순서대로 3개 출력
for (i in 1:3) {
  print(LETTERS[i])
}
## [1] "A"
## [1] "B"
## [1] "C"

다중 반복문은 반복문의 중첩으로 이루어진다. 이때 반복의 순서는 바깥 반복문이 하나씩 반복되는 과정에서 바깥의 각 반복이 이루어지는 동안 안쪽 반복문 전체가 이루어지는 순서로 반복문이 시행된다. 예를 들어 다음의 이중 반복문의 경우 i가 1인 경우 j는 1부터 3까지 반복하며 다시 i가 2인 경우 j는 1부터 3까지 반복하는 순서로 시행된다.

x <- matrix(1:6, nrow = 2, ncol = 3)
x
for (i in 1:2) {
  for (j in 1:3) {
   print(x[i,j])
  }
}

for() 함수를 반복 작업 중에 강제로 종료시키려면 next나 break를 사용한다. next는 현재 진행중인 반복문만 중단하고 다음 반복으로 넘어가며, break는 for() 함수 전체를 종료시킨다.

# 3미만이면 이후 명령을 실행하지 않고 다음으로 넘어가고
# 5이상이면 for 반복에서 빠져 나옴
for (i in 1:10) {
  if (i < 3) {
    next
  }
  print(i)
  if (i >= 5) {
    break
  }
}
## [1] 3
## [1] 4
## [1] 5

for() 함수 내에서 인덱스를 이용해 결과를 저장할때는 사전에 변수(아래 예제에서는 re)를 정의하여야 에러가 나지 않는다. 이는 앞장에서 설명한 모든 객체에 적용이 가능하다. 아래는 객체별 비어있는 형태를 만드는 간단한 방법이다.

vec <- NULL
mat <- matrix(NA, nrow=2, ncol=3)
lis <- list()
a <- c(1, 3, 5)
b <- c(2, 4, 6)
re <- NULL
for (i in 1:3) {
  re[i] <- a[i] + b[i]
}
re
## [1]  3  7 11

아래 예제에서 sum <- sum +i는 재귀식 표현으로 sum이라는 값에 반복이 진행되는 동안 바뀌는 i값을 더한 다음 다시 sum에다 저장을 하게 되고 반복이 진행되는 동안 계속해서 갱신되는 sum 값을 이용하게 된다.

sum <- 0
for (i in 1:100) {
  sum <- sum + i
}
sum

while (조건문) {반복 실행}

6.2.3 repeat()

repeat() 문은 while() 문과 반대로 이용한다. 즉 무조건적으로 반복을 시행하다 조건을 만족하는 순간 반복을 끝내게 된다. 이때 repeat문은 조건을 지정해주는 구문이 없기 때문에 break 문과 같은 명령문을 통해 반복을 제어한다.

repeat { 
  반복 실행
  if (조건) { break }
}
# 1부터 10까지 더하다가 i가 10보다 크면 반복 종료
 sum <- 0
 i <- 1
 repeat {
 sum <- sum + i
 i <- i+1
 if (i > 10) break
 }
 sum
## [1] 55

6.3 사용자 정의 함수

R은 다양한 함수들과 더불어 사용자의 편의에 따라 함수를 정의할 수 있고 이를 통해 프로그램을 간단하고 효율적으로 작정할 수 있다. 사용자가 정의한 함수 또한 고유 형식을 가진 객체이며 함수를 정의하기 위한 형식은 ‘함수이름 <- function(인수들) 명령문’ 이다. 사용자 정의 함수는 항상 제일 마지막에 출력을 위한 명령문이 지정되어야 한다.

함수이름 <- function(인수1, 인수2,...) {
  실행문
  ...
  출력명령
}
f <- function(x) {
  if (x>0) y <- x else y <- -x
  ylog <- log(y)
  ylog
}
f
## function(x) {
##   if (x>0) y <- x else y <- -x
##   ylog <- log(y)
##   ylog
## }
f(5)
## [1] 1.609438

function() 함수는 객체를 함수로 만들어주는 내부 함수이다. 사용자 정의 함수는 항상 할당을 해서 사용해야 한다. f <- function(x){ }의 의미는 f라는 함수 객체를 만든다는 것이며 function() 함수가 이를 가능하게 해준다. 이때 function(x)는 f 라는 함수 내부에서 x라는 인수를 이용해서 처리를 하겠다는 것이다. f라는 함수의 실질적인 내용은 { } 안에서 정의한다. { } 안을 살펴보면 if (x>0) 주어진 인수값 x가 0보다 크면 x 값을 y로 할당을 하고 그게 아니면 –x값을 y로 할당을 한다. 그다음 y값을 log변환해서 ylog에 할당을 하고 마지막으로 ylog값을 출력하라는 것이다. 함수의 마무리는 항상 출력을 위한 명령어가 지정되어야 한다. 그렇지 않으면 함수값이 반환되지 않는다.

s <- function(x) {
  sum <- 0
  for(i in 1:x) {
    sum <- sum + i
  }
  sum
}
s(10)
## [1] 55

R은 함수 안에 또 다른 함수를 사용하는데 자유롭다. 즉, 함수의 인수로 또 다른 함수를 사용해도 되고, 함수를 정의하는 데에도 또 다른 함수들을 중첩시켜도 무방하다.
