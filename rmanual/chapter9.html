9.1 분할표의 재구성

양적(quantitative)으로 관측된 데이터의 경우 위의 함수들을 이용하여 데이터의 기본적인 윤곽을 파악할 수 있으며 질적(qualitative)으로 관측된 데이터의 경우는 도수분포표나 분할표를 이용하여 데이터의 기본적인 특징을 파악할 수 있다.

score <- c(7,7,9,7,10,6,6,7,9,7,9,7,9,10,10,10,6,6,6,8,8,8,7,7)

 n <- length(score)
 par(mfrow=c(2,2))
 barplot(score, col="red")
 barplot(table(score), col="blue")
 barplot(table(score)/n, col="dark red")
 pie(table(score), col=rainbow(24), radius=1)

9.1.1 table()

  • table() : table() 함수는 분할표를 만들어주는 함수이다.
  • rainbow(n) : 빨강-보라까지 색을 n개 만큼 만들어주는 함수이다.
  • with() : attach() 함수와는 달리 일회성으로 데이터를 불러들이는 함수이다.

9.2 가설검정 관련 함수

R에서 기본적으로 제공하는 가설검정 관련 주요 함수는 다음과 같다.

  • t.test() : t-test를 수행하는 함수이다.
  • wilcox.test() : Wilcoxon rank sum test와 signed rank test를 수행하는 함수이다.
  • prop.test() : propotional test를 수행하는 함수이다.
  • binom.test() : binomial test를 수행하는 함수이다.
  • cor.test() : correlation test를 수행하는 함수이다.
  • var.test() : variance test를 수행하는 함수이다.
  • bartlett.test() : Bartlett test를 수행하는 함수이다.
  • kruskal.test() : Kruskal-Wallis test를 수행하는 함수이다.
  • oneway.test() : oneway ANOVA를 수행하는 함수이다.
  • ks.test() : Kolmogorov-Smirnov test를 수행하는 함수이다.
  • shapiro.test() : Shapiro-Wilk test를 수행하는 함수이다.
  • chisq.test() : Pearson chi-squared test를 수행하는 함수이다.
  • fisher.test() : Fisher exact test를 수행하는 함수이다.
  • mcnemar.test() : McNemar’s chi-squared test를 수행하는 함수이다.
