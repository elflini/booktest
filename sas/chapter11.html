※ UPDATE문을 이용한 자료의 갱신

• UPDATE문은 기존의 DATASET을 새로운 정보가 있는 DATASET을 적용하여 갱신 하기 위해 사용된다.

• 이때 갱신하기 위한 기준 변수가 있어야 하고 기준 변수는 BY문 뒤에 사용되며 정렬 이 되어 있어야 하고 두 개 이상의 관찰값을 가져서는 안된다.

 UPDATE 기존 dataset명1 새로운 정보의 dataset명 2 ;
 BY 변수명 ;


11.2 기존의 값에 새로운 값을 추가하기

▶ UPDATE문을 이용하여 DATASET을 바꾸기

<SAS PROGRAM>

data totsales; 
update yrsales weeklist (rename=(sales=newsales)); 
by title; 
sales=sum(sales,newsales); 
drop newsales; 
run; 
proc print data=totsales; 
title 'Updated Year-to-Date Sales' ;
run ;

▶ 특정 변수의 값을 고치는 것 뿐만 아니라 기존의 값에 새로운 값을 추가하여 더하는 것

▶ Totsales DATASET 이용하여 설명
• yrsales DATASET에 weeklist DATASET의 정보를 추가하기
• sales변수의 값을 누적시켜야 하므로 weeklist DATASET의 sales 변수 → newsales
명칭을 바꾼 후 sales=SUM(sales, newsales);라고 하여 합을 구한다.
• sales변수에 다시 넣고 불필요한 newsales변수는 삭제한다.
• sales=sales+newsales라고 하지 않는 이유
: yrsales DATASET의 sales변수에 결측값이 존재하는 경우에 누적된 값도 결측값이 되기 때문이다

<OUTPUT>

                      Updated Year-to-Date Sales                      

 Obs    title                        author               sales

  1     Always Coming Home           LeGuin, Ursula         89
  2     Ender's Game                 Card, Orson Scott      87
  3     Falling through Space        Gilchrist, Ellen      140
  4     The Accidental Tourist       Tyler, Anne            15
  5     The Golden Gate              Seth, Vikran          167
  6     The Handmaid's Tale          Atwood, Margaret       72

▶ MERGE문을 사용하는 경우로 UPDATE문을 사용한 경우와 차이비교

<SAS PROGRAM>

data merged; 
merge master trans; 
by id; 
run ;
proc print data=merged; 
title 'Merging the Master and Transaction Data Sets'; 
title2 'instead of Updating Them' ;
run ;

▶ id변수값이 동일한 것이 존재하는 경우 : UPDATE문은 새로운 값으로 갱신되는 반면 MERGE문은 관찰값이 추가
▶ 차이점
UPDATE : 동일한 변수에 대해 trans DATASET의 변수값이 존재하면 그 값으로 바뀜
MERGE : 새로운 관찰값으로 인식함
