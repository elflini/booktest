Chapter 8 DO - ARRAY

▶ 동일한 작업에 대한 반복 처리를 위해서 SAS에서는 DO문을 사용한다. DO문은 END 문과 짝을 이루어 사용되어진다. 즉, DO문은 END문이 나올 때까지 사이에 있는 여러 개의 명령문들을 실행한다. DO문에는 단순 DO문, 반복을 표시하는 DO문, DO WHILE문 그리고 DO UNTIL문이 있다. 또한 ARRAY문과 결합하여 동일한 작업을 반복할 경우 작업을 간소화한다.

8.1 단순 DO문

<SAS PROGRAM>

data arts; 
input city $ 1-9 museum 11 gallery 13 other 15 guide $ 17-24 yrs_exp 26 ;
cards ;
Rome 4 3 . D'Amico 2 
Paris 5 . 1 Lucas 5 
London 3 2 . Wilson 3 
New York 5 1 2 Lucas 5 
Madrid . . 5 Torres 4 
Amsterdam 3 3 . . 
;
run;
proc print data=arts; 
title 'Data Set ARTS' ;
run ;

▶ DATA의 구성 : city, museum, gallery, other, guide, yrs_exp 등 6개의 변수

<OUTPUT>

                           Data Set ARTS                             

Obs    city         museum    gallery    other     guide     yrs_exp

 1     Rome            4         3         .      D'Amico       2
 2     Paris           5         .         1      Lucas         5
 3     London          3         2         .      Wilson        3
 4     New York        5         1         2      Lucas         5
 5     Madrid          .         .         5      Torres        4
 6     Amsterdam       3         3         .                    .

• 위의 결과는 ARTS DATASET을 출력한 것


▶ 특정 변수의 값이 조건에 맞는 경우에 다수 개의 변수의 값을 바꾸는 것으로 IF문과 DO, END문 사용

<SAS PROGRAM>

data fixit2; 
set arts; 
if city='Madrid' then 
do ;
museum=3 ;
other=2 ;
end ;
else if city='Amsterdam' then 
do ;
guide='Vandever' ;
yrs_exp=4 ;
end ;
run ;
proc print data=fixit2; 
title 'Using DO Groups' ;
run ;

▶ city 변수의 값이 ‘Madrid’이면 museum값은 3, other변수값은 2로 바꾸고, ‘Amsterdam’이면 guide는 ‘Vandever’이고 yrs_exp는 4로 바꾼다.

<OUTPUT>

                          Using DO Groups                            

 Obs    city         museum    gallery    other    guide       yrs_exp

  1     Rome            4         3         .      D'Amico        2
  2     Paris           5         .         1      Lucas          5
  3     London          3         2         .      Wilson         3
  4     New York        5         1         2      Lucas          5
  5     Madrid          3         .         2      Torres         4
  6     Amsterdam       3         3         .      Vandever       4

• city변수값이 ‘Madrid’인 5번째 관찰값 museum, other변수값이 바뀌고, ’Amsterdam’인 6번째 관찰값중 guide, yrs_exp변수값이 바뀌었다.

※ IF 문 사용하기 6 - 실행문이 여러 개일 경우 (DO문과 END문)

• IF문의 사용시 하나의 조건에 맞는 경우 두 개 이상의 실행문을 사용하고자 할 경우 DO문과 END문을 사용

IF 조건문 1 THEN DO ;
    실행문 ;
           ;
    실행문 ;
   END;         ⟹ 조건문 1이 참이면 실행문 에서 실행문 까지 수행한다.
ELSE IF 조건문 2 THEN DO;
    실행문 ;
          ;
    실행문 ;
   END;         ⟹ 조건문 2가 참이면 실행문 에서 실행문 까지 수행한다.


8.2 반복 DO문

▶ 반복 DO 문은 DO 문안에 색인을 나타내는 색인(index)변수를 지정하고, 색인 변수의 값을 증가시켜 색인 변수의 최종 값과 일치하면 반복을 중지한다. 증가 값은 기본적으로 1이지만 BY에 의해 증가 값을 지정할 수도 있다.

<SAS PROGRAM>

data do;
k=20;
do i=1 to 8 ;
  ke=k +.8 + i;
  end;
  run;

▶ i는 1부터 8까지 1단위로 증가하면서 값이 바뀌며, 1부터 8까지의 크기인 8만큼 반복 수행하게 된다.
▶ ke변수는 k의 값에 i이 변함에 따라 반복해서 더해지게 된다.
▶ 결과는 8번의 반복이 끝난 후 i는 8 다음 단위인 9가 표시되며, ke변수는 마지막 반복값이 저장된다.

<OUTPUT>

                            Using DO

                     OBS     k    i     ke
                      1     20    9    28.8

• i는 9이며 ke는 마지막 반복값이 28.8이 됨을 알 수 있다.

※ 반복 DO문

  DO 색인변수=시작 값 [TO=최종 값 [BY 증가분] [WHILE 또는 UNTIL(표현식)]];
   SAS 문장들;
  END;

단, [ ]는 반드시 사용하지 않아도 된다는 것을 나타낸다.


▶ DO 루프의 단계별 생성값의 저장(output)

<SAS PROGRAM>

data do;
k=20;
do i=1 to 8 ;
  ke=k +.8 + i;
output;
  end;
  run;

▶ DO문과 END문 사이에 OUTPUT문을 넣으면 반복 결과를 모두 볼 수 있다.

<OUTPUT>

                            Using DO

                     OBS     k    i     ke
                      1     20    1    21.8
                      2     20    2    22.8
                      3     20    3    23.8
                      4     20    4    24.8
                      5     20    5    25.8
                      6     20    6    26.8
                      7     20    7    27.8
                      8     20    8    28.8

• 각 반복단계의 결과를 알 수 있다.


8.3 다중 반복 DO문

<SAS PROGRAM>

data do6(drop=i);
 k=20;
  do i=1 to 20 ;
   k=k+100;
     do j=1 to 10;
        ke=k +.8 + j;
      output;
     end;
  end;
  run;

▶ i는 1부터 20까지 1단위로 증가하면서 값이 바뀌며, 1부터 20까지의 크기인 20만큼 반복 수행하게 된다.

▶ j는 1부터 10까지 1단위로 증가하면서 값이 바뀌며, 10만큼 반복수행 하게 된다.

▶ 이때 반복수행 순서는 i의 값이 바뀔 때마다 j는 전체 반복을 한다. 즉, i=1일 때 j는 1부터 10까지 반복하며 다시 i=2 일 때 j는 1부터 10까지 반복한다. 다시 말해서 반복 순서는 바깥쪽에서부터 색인의 값이 하나씩 결정되며 제일 안쪽에 있는 DO문은 전체를 반복한다. 안쪽의 DO문이 끝나면 바깥쪽으로 순서대로 DO문을 끝낸다.

                     4    120     4    124.8

8.4 조건부 DO문

▶ 조건부 DO문은 반복 횟수를 알지 못하는 경우 사용한다.

▶ DO UNTIL과 DO WHILE이 여기에 해당한다.

<SAS PROGRAM>

data do7;
  k=20;
  do until(ke>=200);
   k=k+10;
    ke=k +.8;
    n+1;
    output;
  end;
  run;

data do8;
  k=20;
  do while(ke<=200);
   k=k+10;
    ke=k +.8;
    n+1;
    output;
  end;
  run;

▶ DO UNTIL은 UNTIL 뒤에 나오는 표현식이 참이 될 때까지 반복 수행한다. 즉, ke값이 200이상이 되는 순간 반복을 멈춘다.

▶ DO WHILE은 WHILE 뒤에 나오는 표현식이 참인 동안에만 반복 수행한다. 즉, ke값이 200이하인 동안에만 반복을 수행하며, 다시 말해서 KE가 200을 초과하는 순간 반복을 멈춘다.

※ DO UNTIL문

Do Until(표현식);
   문장;
end;

※ DO WHILE문

Do While(표현식);
   문장;
end;


8.5 ARRAY문의 활용

▶ 2개 이상이ㅡ 변수에 동일한 처리 할 경우 DO문과 ARRAY문 사용

<SAS PROGRAM>

data changes; 
set arts; 
array chglist{3} museum gallery other; 
do count=1 to 3 ;
if chglist{count}=. then chglist{count}=0 ;
end ;
run ;
proc print data=changes; 
title 'Data Set Produced with Array Processing' ;
run ;

▶ museum, gallery, other 변수값이 결측값을 가질 경우 전부 0으로 교환.

<OUTPUT>

             Data Set Produced with Array Processing                

 Obs   city        museum   gallery   other    guide    yrs_exp   count

  1    Rome           4        3        0     D'Amico      2        4
  2    Paris          5        0        1     Lucas        5        4
  3    London         3        2        0     Wilson       3        4
  4    New York       5        1        2     Lucas        5        4
  5    Madrid         0        0        5     Torres       4        4
  6    Amsterdam      3        3        0                  .        4

• museum, gallery, other의 결측값이 전부 0으로 바뀜.
• 반복수행을 위하여 사용되었던 count변수가 첨가되어 있으며 전부 4의 값을 가짐.
• DO문의 수행은 거짓이 될 때까지(3까지만 수행하도록 하였으므로 1이 증가한 4는 거짓이 되지만 count값은 4까지 변함) 수행하는 것을 볼 수 있다.


▶ DATASET을 형성하는데 불필요하면서, DO문을 반복수행하기 위해 생성된 count변수 제거

<SAS PROGRAM>

data chgdrop(drop=count); 
   set arts; 
   array chglist{3} museum gallery other; 
   do count=1 to 3 ;
      if chglist{count}=. then chglist{count}=0 ;
   end ;
run ;
proc print data=chgdrop; 
   title 'Data Set After Dropping Index Variable' ;
run ;

▶ ARRAY문을 사용할 경우에 반복 수행을 위해 사용되어진 변수는 제거하는 것이 좋음.

<OUTPUT>

               Data Set After Dropping Index Variable                8

Obs    city         museum    gallery    other     guide     yrs_exp

 1     Rome            4         3         0      D'Amico       2
 2     Paris           5         0         1      Lucas         5
 3     London          3         2         0      Wilson        3
 4     New York        5         1         2      Lucas         5
 5     Madrid          0         0         5      Torres        4
 6     Amsterdam       3         3         0                    .

• 불필요한 count 변수를 제거한 후에 실제로 원하는 DATASET을 형성하여 출력.
