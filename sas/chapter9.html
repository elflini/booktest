Chapter 9 날짜 자료

▶ SAS시스템은 1960년 1월 1일을 기준으로 숫자값으로 날짜 시간, datetime을 저장한다.

9.1 DATE 문

<SAS PROGRAM>

data dates;
 input country $ 1-11 @13 depart date7. nights;
cards;
Japan       13may89 8
Greece      17oct89 12
New Zealand 03feb90 16
Brazil      28feb90 8
Venezuela   10nov89 9
; run;
proc print data=dates;
run;

data dates_2;
set dates;
format depart date7.;
run;
proc print data=dates_2;
run;


proc print data=dates;
format depart mmddyy8.;
run;

▶ input 문장에서 날짜 자료는 위의 형식과 같이 ‘date자릿수.’ 형식을 이용한다.

▶ 형식은 자료의 형태마다 달라진다.

[날짜 형식과 출력형식의 관계]
- MMDDYYw. : (예. MMDDYY6. → 102301, MMDDYY8. → 10/23/01, MMDDYY10. → 10/23/2001)
- DDMMYYw.
- YYMMDDw. : (예.YYMMDD6. → 011023, YYMMDD8. → 01-10-23, YYMMDD10. → 2001-10-23)
- YYMMDDnw. : no separator (예.YYMMDDn8. → 20011023)
- YYMMDDpw. : period (예.YYMMDDp8. → 01.10.23, YYMMDDp10. → 2001.10.23)
- YYMMDDsw. : slash(/)
- YYMMDDcw. : colon(:)
- YYMMDDdw. : dash(-) - DATE7. : 일 월(영문3자리) 년 (05OCT88)
- WORDDATE. : 영문월 일, 년(OCTOBER 05, 88)

9.2 날짜 계산

 <SAS PROGRAM>
 
 data date_cal;
  set dates;
  new=depart+nights;
  format new date7.;
  run;

  data date_dif;
  set date_cal;
  dif=intck('day',depart,new);
  dif1=intck('week',depart,new);
  run;

  data date_dd;
   date = INTCK ('year','31dec89'd,'1jan90'd);
   year = INTCK ('year','1jan89'd,'31dec89'd);
   mon1 = put(INTNX('month','23oct01'd,5,'beginning'),yymmddn8.);
   mon2 = put(INTNX ('month','23oct01'd,5,'middle'),yymmddn8.);
   mon3 = put(INTNX ('month','23oct01'd,5,'end'),yymmddn8.);
  run;

▶ INTNX 함수 설명
• mon1 = put(INTNX(‘month,’‘23oct01’d,5,’beginning’),yymmddn8.);
특정시점 2001년 10월23일로부터 5개월 후의 처음 시작 일자를 생성함.
즉 2001년10월23일에서 5개월 이후는 2002년 3월인데 그 중에 beginning 라는 문장이 있기 때문에 2002년 3월에서 첫 일자를 선택한다.

• mon2 = put(INTNX (‘month,’‘23oct01’d,5,’middle’),yymmddn8.);
특정시점 2001년 10월23일로부터 5개월 후의 중간 일자를 생성함.
즉 2001년10월23일에서 5개월 이후는 2002년 3월인데 그 중에 middle 라는 문장이 있기 때문에 2002년 3월에서 중간 일자를 선택한다.

• mon3 = put(INTNX (‘month,’‘23oct01’d,5,’end’),yymmddn8.);
특정시점 2001년 10월23일로부터 5개월 후의 마지막 일자를 생성함.
즉 2001년10월23일에서 5개월 이후는 2002년 3월인데 그 중에 end 는 문장이 있기 때문에 2002년 3월에서 마지막 일자를 선택.

  • 사용하는 목적: 어느 그룹에서 어떤 특정한 값을 산출 하거나 특정 시점에서 작업을 원할 경우 필요함.

▶ INTCK 함수 프로그램 설명
• date = INTCK (‘year,’’31dec89’d,’1jan90’d);
특정시점 1989년12월31일에서 1990년1월1일까지 걸쳐진 년도를 생성함.
즉 1990-1989 이므로 1 이란 값이 생성.

• year = INTCK (‘year,’’1jan89’d,’31dec89’d);
특정시점 1989년1월1일부터 1989년12월31이므로 즉 1989-1989 이므로 0 이란 값이 생성됨

• INTCK 함수에서 YEAR , MONTH , QTR, DAY , WEEK 는 구하고자 하는 용도에 따라 사용됨.

  • 사용목적: 어느 그룹에서 과거나 미래의 어떤 특정한 시점에서 달이나 년이나 월이나 분기를 구하고자 할 때 사용됨.
